# 烁烁南光在线免费影视

## 项目介绍

烁烁南光在线免费影视是一个基于Vue 3开发的影视在线播放平台，提供电影、电视剧、动漫、短剧、综艺等内容的在线观看服务。

## 技术栈

- Vue 3
- Vue Router
- Pinia
- Element Plus
- Vite

## 功能特点

- 响应式设计，适配多种设备
- 黑暗模式/亮色模式切换
- 侧边栏可折叠
- 分类浏览
- 搜索功能
- 观看历史记录
- 收藏功能

## 安装与运行

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```
