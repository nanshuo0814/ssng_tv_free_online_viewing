## 项目介绍

烁烁南光在线免费影视是一个基于Vue 3开发的影视在线播放平台，提供电影、电视剧、动漫、短剧、综艺等内容的在线观看服务。平台采用现代化的前端技术栈，提供流畅的用户体验和丰富的功能特性。

## 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- Vue Router - 官方路由管理器
- Pinia - Vue 的状态管理库
- Element Plus - 基于 Vue 3 的组件库
- Vite - 下一代前端构建工具
- Axios - 基于 Promise 的 HTTP 客户端
- VueUse - Vue Composition API 的实用工具集合

## 功能特点

- 响应式设计，完美适配多种设备（PC、平板、手机）
- 黑暗模式/亮色模式切换，保护用户视力
- 视频画中画功能，支持多任务处理
- 视频播放进度记忆，下次观看自动续播
- 智能搜索功能，支持模糊匹配
- 分类浏览，快速找到感兴趣的内容
- 收藏功能，方便下次观看
- 播放历史记录，随时回顾
- 视频清晰度切换，满足不同网络环境需求

## 项目结构

```
src/
├── assets/          # 静态资源
├── components/      # 公共组件
├── router/          # 路由配置
├── stores/          # Pinia 状态管理
├── styles/          # 全局样式
├── utils/           # 工具函数
└── views/           # 页面组件
```

## 安装与运行

1. 克隆项目
```bash
git clone https://github.com/nanshuo0814/ssng_tv_free_online_viewing.git
cd ssng_tv_free_online_viewing
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```
